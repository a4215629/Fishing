# 浮漂走水受力分析仿真

钓鱼浮漂在水流（走水）作用下的受力分析与交互式仿真。

## 项目简介

在河流中钓鱼时，水流会冲击浮漂和鱼线，导致浮漂被拉入水中（走水）。本项目通过物理建模和可视化仿真，分析浮漂-鱼线-铅坠系统在水流作用下的受力平衡与动态过程。

## 物理模型

### 核心公式

- **θ**：鱼线与水平面的夹角
- **p**：浮漂浸入水中的比例，由几何约束 `p = clamp(1 - ((L_l+L_f)×sinθ - D) / (L_f×sinθ), 0, 1)` 决定
- **平衡条件**：`tanθ = Ff / Fc`（浮力与总冲力之比）

### 受力分解（以鱼线方向为基准轴）

| 力 | 公式 | 方向 |
|---|------|------|
| 浮力 Ff | `Ff_max × p` | 竖直向上 |
| 上升力 Fs | `Ff × cosθ` | 垂直鱼线向上 |
| 下压力 Fy | `Fc × sinθ` | 垂直鱼线向下 |
| 浮漂冲力 Fc_f | `0.5 × ρ × v² × Cd_f × d_f × L_f × p × sinθ` | 水平 |
| 鱼线冲力 Fc_l | `0.5 × ρ × v² × Cd_l × d_l × L_l × sinθ` | 水平 |
| 线拉力 Fl | `|Ff×sinθ - Fc×cosθ|` | 沿鱼线方向 |

### 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 浮漂长度 L_f | 20 cm | 圆柱形浮漂 |
| 排水量 | 3.2 g | 吃铅 3g |
| 浮漂直径 d_f | 0.45 cm | |
| 鱼线号数 | 1.5 号 | 直径自动换算 |
| 鱼线直径 d_l | 0.0205 cm | 1.5号对应值 |
| 鱼线长度 L_l | 140 cm | 水深方向 |
| 锚点水深 D | 150 cm | 铅坠位置 |
| 水流速度 v | 0.1 m/s | 可调 |
| 浮漂阻力系数 Cd_f | 0.4 | 纺锤形侧面迎流 |
| 鱼线阻力系数 Cd_l | 0.7 | 细圆柱含弯曲折减 |

## 使用方法

直接用浏览器打开 `simulation.html`，无需任何构建工具或服务器。

### 功能

- **参数调节**：浮漂、鱼线、水流、阻力系数等参数实时可调
- **动画仿真**：基于半隐式欧拉积分的物理动力学模拟
- **实时数据**：角度、浸入比例、各力大小的实时显示
- **力-时间曲线**：7 条力曲线，可点击图例切换显示
- **多主题**：深海蓝、明亮白、高对比、暖色、自然绿 5 种配色
- **移动端适配**：支持触屏操作，参数/仿真 Tab 切换

### 验证脚本

```bash
node validate.js
```

检查 JS 语法、主题变量定义、变量命名冲突等。

## 技术栈

- 纯 HTML + CSS + JavaScript（ES6），零依赖
- Canvas 2D API 绘制动画、力矢量、图表、装饰鱼
- Flexbox 响应式布局，支持桌面和移动端

## 参考资料

- 阻力公式：`F_drag = 0.5 × ρ × v² × Cd × A`
- 鱼线号数-直径换算：`d(mm) = 0.05 × √(号数 × 10)`
- 浮漂阻力系数 Cd_f ≈ 0.3~0.5（纺锤形侧面迎流）
- 鱼线阻力系数 Cd_l ≈ 0.5~0.8（细圆柱横向绕流，含弯曲折减）
